# Log-File-Analyzer-Python
A Python script to analyze log files and generate reports

This is a simple Python automation script that analyzes log files
and generates a summary report.

## Features
- Reads log files automatically
- Counts ERROR, WARNING, and INFO logs
- Generates a clean report file
- Beginner-friendly Python project

## How to Run
1. Make sure Python is installed
2. Place your logs inside `logs.txt`
3. Run:
   python analyzer.py
4. Check `report.txt` for output

## Use Cases
- Server log analysis
- Application monitoring
- Automation practice
